Install

Binaries and packages ship for Linux, macOS, and Windows on a variety of architectures. Every tagged release publishes archives and platform packages to GitHub Releases.

GitHub Releases all platforms

Download the archive for your OS and architecture from the latest release, extract it, and place nginxfmt on your PATH.

Homebrew macOS / Linux

brew tap nginxfmt/homebrew-tap
brew install --cask nginxfmt

Cargo Rust

cargo install nginxfmt

Published to crates.io on each release. Requires a Rust toolchain — grab one at rustup.rs.

Debian / Ubuntu .deb

sudo dpkg -i nginxfmt_*_amd64.deb

Download the .deb for your architecture from the latest release.

Linux packages Fedora / Alpine / Arch

.rpm, .apk, and Arch Linux packages are attached to each GitHub release.

Scoop Windows

scoop bucket add nginxfmt https://github.com/nginxfmt/scoop-bucket
scoop install nginxfmt

Winget Windows

winget install nginxfmt.nginxfmt

Manifests are published to nginxfmt/winget on each release.

Chocolatey Windows

choco install nginxfmt

.nupkg files are also attached to GitHub Releases.

Usage

Point it at a file. By default it formats and prints to stdout.

Format & print to stdout

nginxfmt conf/nginx.conf

Format in place

nginxfmt --write conf/nginx.conf

Check formatting (CI)

nginxfmt --check conf/nginx.conf

Read from stdin (unix)

cat conf/nginx.conf | nginxfmt -

Configuration

Drop a .nginxfmt.toml in your project. It's auto-discovered by walking up from the input file. CLI flags override file values.

.nginxfmt.toml

indent_style = "spaces"   # or "tabs"
indent_width = 4
brace_style = "same_line" # or "next_line"
max_blank_lines = 1
trailing_newline = true
preserve_inline_comments = true

Override with flags

nginxfmt \
  --tabs \
  --indent-width 2 \
  --brace-style next_line \
  conf/nginx.conf

Options

Every flag, and what it defaults to.

Option Description Default
--write, -w Write formatted output back to the file stdout
--check Exit 1 if input is not formatted off
--config PATH Explicit config file path auto-discover
--tabs / --spaces Indentation style spaces
--indent-width N Spaces per indent level 4
--brace-style same_line or next_line same_line
--max-blank-lines N Collapse extra blank lines 1
--trailing-newline Ensure trailing newline true
--preserve-inline-comments Keep # comments on same line true
